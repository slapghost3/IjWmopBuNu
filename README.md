# 项目简介

该项目为基于Python实现的简易博客系统，包含源码和SQL数据库文件，旨在提供一个基础的博客发布与管理系统。

## 核心功能

- 用户管理：登录注册，个人信息修改
- 文章管理：编辑文章发布，文章评论，文章信息管理
- 分类管理：文章分类的创建和管理

## 适用场景

适用于个人博客搭建、团队内部知识分享、小型企业或组织的信息发布平台。

## 技术特点

- 使用Django框架，基于Python实现，具备良好的扩展性和维护性
- 采用MySQL数据库进行数据存储，保证数据安全和稳定性
- 前端界面采用HTML、CSS和JavaScript等技术，提供良好的用户体验
